
# ACA2 - Herramientas de Big Data

Proyecto académico orientado al análisis de contratación pública mediante técnicas de Big Data y Machine Learning utilizando PySpark y Spark MLlib.

## Objetivo

Predecir la concentración de proveedores en procesos de contratación pública utilizando modelos de clasificación supervisada aplicados sobre datos del SECOP.

## Tecnologías utilizadas

- Python
- PySpark
- Spark MLlib
- Jupyter Notebook
- Power BI
- GitHub

## Estructura del proyecto

```bash
data/           # Datos originales y procesados
notebooks/      # Notebook principal
src/            # Scripts auxiliares
dashboard/      # Visualizaciones Power BI
docs/           # Documento final
results/        # Resultados y métricas
```
